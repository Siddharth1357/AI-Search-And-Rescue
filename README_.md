# AI-Powered Multi-Modal Search & Rescue System

## Overview
This project is a low-cost disaster response system designed to detect trapped survivors using multiple sensing modalities.

## Features
- Radar-based motion detection
- Camera vision system
- Thermal detection module
- Audio monitoring
- CNN-based Edge AI inference
- ESP32-S3 deployment
- Wireless alert generation
- Low-power reactive sensing architecture

## Hardware
- ESP32-S3
- Camera Module
- Thermal Sensor
- Microphone
- Radar Sensor

## Applications
- Earthquake rescue
- Building collapse response
- Landslide rescue
- Mine accident detection

## Future Scope
- Drone integration
- Thermal imaging enhancement
- Mesh networking
- Autonomous rescue robots
